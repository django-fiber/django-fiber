jQuery UI Pulsate Effect

Overview

Pulsates the opacity of the element multiple times.

Dependencies

  • Effects Core

Example

  • Demo
  • View Source

Pulsate 3 times when clicking on the element.

$("div").click(function () {
      $(this).effect("pulsate", { times:3 }, 2000);
});

Arguments

  • mode

    Type:
    String
    Default:
    "show"

    The mode of the effect. Can be "show" or "hide".

  • times

    Type:
    Integer
    Default:
    5

    Times to pulsate.
