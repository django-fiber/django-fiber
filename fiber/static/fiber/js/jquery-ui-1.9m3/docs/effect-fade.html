jQuery UI Fade Effect

Overview

Fades the element. The same effect as jQuery's fadeIn/Out/To methods, but exposes it to the show, hide and effect methods.

Dependencies

  • Effects Core

Example

  • Demo
  • View Source

Hides the element by fading it out.

$("div").click(function () {
      $(this).hide("fade", {}, 1000);
});
